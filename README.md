## Запуск автотестов

```bash
# Перейти в папку task_2
cd task_2

# Установка зависимостей
pip install -r requirements.txt

# Запуск тестов
pytest test_api.py -v
```

Подробная инструкция по запуску находится в [task_2/README.md](task_2/README.md).

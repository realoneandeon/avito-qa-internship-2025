## Описание заданий

Данный репозиторий содержит решения тестового задания для стажёра QA-направления (зимняя волна 2025) от Авито.

### Структура репозитория

```
.
├── README.md                    # Общее описание проекта
├── task_1/                      # Задание 1: Анализ багов по скриншоту
│   └── bugs.md                  # Список найденных багов
└── task_2/                      # Задание 2.1: API тестирование
    ├── TESTCASES.md             # Тест-кейсы для API микросервиса
    ├── test_api.py              # Автоматизированные тесты для API
    ├── requirements.txt         # Зависимости проекта
    ├── README.md                # Инструкция по запуску тестов
    └── BUGS.md                  # Баг-репорты (заполняется после тестирования)
```

## Запуск автотестов

```bash
# Перейти в папку task_2
cd task_2

# Установка зависимостей
pip install -r requirements.txt

# Запуск тестов
pytest test_api.py -v
```

Подробная инструкция по запуску находится в [task_2/README.md](task_2/README.md).
